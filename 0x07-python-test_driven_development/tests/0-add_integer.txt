The ``Add Integer`` module
============================

Using ``add_integer``
---------------------

First ``add_integer`` from the ``Add Integer`` module:

    >>> add_integer = __import__("0-add_integer").add_integer

Now use it:

    >>> add_integer(1, 1)
    2

    >>> add_integer(1, 2.2)
    3

    >>> add_integer(2.2, 1)
    3

    >>> add_integer(2.2, 2.2)
    4
